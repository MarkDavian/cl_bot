Обработчики команд (Handlers)
=============================

Обработчики команд Telegram-бота для различных разделов функциональности.

Команды меню
==============

Команды для работы с главным меню бота:

**cmd_start(message)** - Отображает главное меню бота
  * Создает и отправляет сообщение с кнопками основных разделов
  * Регистрирует обработчики нажатий на кнопки

**process_menu_selection(callback_query)** - Обрабатывает выбор пункта меню
  * Определяет выбранный пункт и переходит к соответствующему разделу

Управление сотрудниками
=======================

Команды для базового управления данными сотрудников:

**cmd_add(message)** - Запускает процесс добавления нового сотрудника
  * Переводит бота в состояние ожидания имени сотрудника
  * Последовательно запрашивает все необходимые данные

**cmd_emps(message)** - Отображает список всех сотрудников
  * Формирует и отправляет список сотрудников с ключевой информацией
  * Добавляет кнопки для управления отдельными сотрудниками


Настройки
==========

Команды для управления настройками бота:

**cmd_settings(message)** - Отображает текущие настройки бота
  * Показывает основные параметры конфигурации
  * Предоставляет возможность их изменения
