Модуль Backup
=============

Система резервного копирования и экспорта данных.

Основной модуль резервного копирования
=======================================

Функции модуля:

**backup()** - Асинхронная функция для создания и отправки резервных копий данных.
  
  Выполняет следующие операции:
  
  1. Создает Excel-файл с несколькими листами, содержащими разные представления данных о сотрудниках
  2. Применяет форматирование и стилизацию к созданным таблицам
  3. Создает ZIP-архив директории storage, содержащей все данные приложения
  4. Отправляет созданные файлы в указанный Telegram-чат
  5. Повторяет процесс через заданный интервал времени (по умолчанию - 24 часа)

**start_bot()** - Асинхронная функция для запуска бота.
  
  Инициализирует бота с указанным токеном и запускает обработку сообщений.
