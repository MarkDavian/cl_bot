Модуль Notifier
=======================

Система уведомлений и проверки значимых событий.

.. toctree::
   :maxdepth: 2
   
   checkers
   
Основные компоненты модуля:

**check_events_periodically()** - Асинхронная функция, которая периодически проверяет различные события и отправляет уведомления
  * Проверяет медицинские книжки (сроки действия)
  * Проверяет дни рождения сотрудников
  * Проверяет юбилеи (стаж работы)
  * Проверяет сертификаты
  * Проверяет состояние кофемашин

**start_bot()** - Асинхронная функция для запуска бота уведомлений
  * Настраивает бота и обработку сообщений
  * Запускает периодическую проверку событий
